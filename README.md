# SEVA-Connect---Wireframe
mobile‑first, multilingual (Sinhala/Tamil/English), WCAG 2.1 AA–aligned UI flows for diaspora volunteers and Sri Lankan government (DS) needs, including verification tiers, matching, Terms of Engagement (ToE), milestone tracking, and impact dashboards.
This repository contains an **HTML wireframe** for the **SEVĀ Connect** digital platform.  
It shows the main user flows between **diaspora volunteers**, the **SEVĀ Connect system**, and **Divisional Secretariat (DS) / government** users.


## How to open
1. Download or clone this repo
2. Open the HTML file in Chrome/Edge/Firefox  
   (double-click the file, or right-click → Open with)

## What the wireframe includes
- DS creates and publishes project/needs listings
- Volunteer registration + document upload
- System verification gate (tiers)
- Volunteer discovers projects and applies (solo/group)
- DS reviews and interviews (can be off-platform)
- Terms of Engagement (ToE), milestone tracking, and impact reporting (concept)
